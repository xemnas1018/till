# @using(ユージング)  

## 使用箇所  
* 他のディレクトリに保管されている情報の引用先ファイル
  
## 解説  
* 名前空間と呼ばれる各種データや機能が整理されている場所から特定の機能やデータを引用するディレクティブ。      
  
* メソッドやクラス、予めBlazerに搭載されている機能等を引用先で使用することができる。

* 名前空間はクラスとの差別化の為、部分的な頭文字大文字のキャメルケースで定義する。例：MyNamespace
  
## 雛形   
```C#
  // 名前空間の引用
  @using [名前空間]

  // 名前空間の定義
  namespace 名前空間名
  {
　// 名前空間に含めるクラスやメソッドを定義
  }
```

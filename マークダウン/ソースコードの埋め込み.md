# ソースコードの埋め込み  
## 解説  
* 文字列ではなく、ソースコードとして表示したい箇所を```で囲むことでソースコードの埋め込みができる。  

* この状態で記述すると通常のテキストエディタと同様の記法で正常に表示させることが可能になり、
  コードのコピーボタンが自動で生成される。

* 始点となる```の後ろに各コードを示す単語を追記するとその言語に応じた表示をしてくれる。
   
|言語名|記述する単語|
|:-------:|:--------:|
|Ruby     |ruby又はrb |
|``<br>``タグ|``<br>``を改行したい行数分入力|上限無し|


## 雛形  
```
改行したい␣␣
改行したい<br>
```

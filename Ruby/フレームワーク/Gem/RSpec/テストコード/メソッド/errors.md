# include(インクルード)  
## 解説  
●インスタンスにエラーを示す情報がある場合にその内容を返すメソッド。  

●確認する際は、``rails c``を用いてターミナルでエラーが発生する処理を行った後に、``errors``を入力する。  

●複雑なエラー内容を直接``expect``に記載することはできない為、  
``full_messagesメソッド``を使用してエラー内容を取得する必要がある。

## 雛形 
```bush
エラー内容を確認したいインスタンス.errors
=> エラー内容が表示される
```
## 関連ワード  
●``full_messages``
